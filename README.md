**Terraform ignore file**

`.terraform/` — будут игнорироваться все файлы и папки внутри директории .terraform (сама директория и всё её содержимое).

`*.tfstate` — будут игнорироваться все файлы с расширением .tfstate.

`*.tfstate.*` — будут игнорироваться все файлы, имя которых начинается с любых символов, затем содержит .tfstate., после чего следует любые другие символы.

`crash.log` — будет игнорироваться файл с точным именем crash.log.

`crash.*.log` — будут игнорироваться все файлы, имя которых начинается с crash., затем содержит любые символы и заканчивается на .log.

`*.tfvars` — будут игнорироваться все файлы с расширением .tfvars.

`*.tfvars.json` — будут игнорироваться все файлы с расширением .tfvars.json.

`override.tf` — будет игнорироваться файл с точным именем override.tf.

`override.tf.json` — будет игнорироваться файл с точным именем override.tf.json.

`*_override.tf` — будут игнорироваться все файлы, имя которых заканчивается на _override.tf.

`*_override.tf.json` — будут игнорироваться все файлы, имя которых заканчивается на _override.tf.json.

`.terraform.tfstate.lock.info` — будет игнорироваться файл с точным именем .terraform.tfstate.lock.info.

`.terraformrc` — будет игнорироваться файл с точным именем .terraformrc.

`terraform.rc` — будет игнорироваться файл с точным именем terraform.rc
